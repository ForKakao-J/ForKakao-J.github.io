MYSQL的基本操作

一、字段的类型和属性

整形

  1. MySQL数据类型
  2. tinyint(m)
  3. smallint(m)
  4. mediumint(m)
  5. int(m)
  6. bigint(m)
  • 含义
  • 1个字节 范围(-128~127)
  • 2个字节 范围(-32768~32767)
  • 3个字节 范围(-8388608~8388607)
  • 4个字节 范围(-2147483648~2147483647)
  • 8个字节 范围(+-9.22*10的18次方)

浮点型

  1. MySQL数据类型
  2. float(m,d)
  3. double(m,d)
  • 含义
  • 单精度浮点型 8位精度(4字节) m总个数，d小数位
  • 双精度浮点型 16位精度(8字节) m总个数，d小数位

字符串

  1. MySQL数据类型
  2. char(n)
  3. varchar(n)
  4. tinytext
  5. text
  6. mediumtext
  7. longtext
  • 含义
  • 固定长度，最多255个字符
  • 固定长度，最多65535个字符
  • 可变长度，最多255个字符
  • 可变长度，最多65535个字符
  • 可变长度，最多2的24次方-1个字符
  • 可变长度，最多2的32次方-1个字符

日期时间类型

  1. MySQL数据类型
  2. date
  3. time
  4. datetime
  5. timestamp
  • 含义
  • 日期 '2019-9-17'
  • 时间 '19:25:36'
  • 日期时间 '2019-9-17 19:25:44'
  • 自动存储记录修改时间

类型的属性

  1. MySQL关键字
  2. NULL
  3. NOT NULL
  4. DEFAULT
  5. PRIMARY KEY
  6. AUTO_INCREMENT
  7. UNSIGNED
  8. CHARACTER SET name
  • 含义
  • 数据列可包含NULL值
  • 数据列不允许包含NULL值
  • 默认值
  • 主键(不可重复)
  • 自动递增，适用于整数类型
  • 无符号
  • 指定一个字符集

二、基本控制语句

  1. 创建表
    • create table [表名]([字段名] 类型 , [字段名] 类型... , 属性)

  2. 复制表
    • create table [table1] select * from [table2];
      • 或者部分复制: 　　　　
    • create table [table1] select id,name from [table2];

  3. 查看数据库中可用的表
    • show tables;

  4. 查看表的结构
    • describe [table];
      • 或者: 　　　　
    • show columns from [table];

  5. 删除表
    • 删表很简单，操作须谨慎^.^
    • drop table [table];

  6. 表重命名
    • alter table [old] rename [new];

  7. 更改表结构
    • 修改字段名: alter table [old] rename [new];
    • 添加表列: alter table [table] add column [字段名] varchar(10);
    • 修改表列类型: alter table [table] modify [字段名] char(10);
    • 修改表列名: alter table [table] column [字段名] [字段名new] varchar(30);

  8. 插入数据
    • insert into [table](id,name,score)VALUES(NULL,'张三',140),(NULL,'张四',178),(NULL,'张五',134);

  9. 更新数据
    • update [table] set [修改的值] WHERE [条件/筛选];

  10. 删除数据
    • delete from [table] where [条件/筛选];

  11. 条件控制
    • delete from [table] where [条件/筛选];

三、多表查询

  1. 内连接
    • select [table].[字段] , [table2].[字段] from [table] inner join [table2] on [table].[字段] = [table2].[字段] ;

  2. 左连接
    • select [table].[字段] , [table2].[字段] from [table] left join [table2] on [table].[字段] = [table2].[字段] ;

  3. 右连接
    • select [table].[字段] , [table2].[字段] from [table] right join [table2] on [table].[字段] = [table2].[字段] ;

  4. 完全连接
    • select [table].[字段] , [table2].[字段] from [table] full join [table2] on [table].[字段] = [table2].[字段] ;

四、视图

    创建视图
    • create view [视图名] as select [table].[字段] as [新字段] from [table];
